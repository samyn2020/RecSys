# RecSys
Recommends whether or not a user should watch movie(s) based on what other users watched
____________
Dependencies  
---------------
* Python  
  * https://www.python.org/  
____________
Notes  
---------------
* Both programs have user arrays at the top (edit them)  
  * 0 = did not watch  
  * 1 = did not watch  
  * 2 = needs recommendation
